Html标签类型分类

块级标签

特点：独占一行，宽高可以改变（如：div，h）

行内标签

特点：不独占一行，宽高不可以改变（如：span，img，a）

行内块级标签

特点：不独占一行，宽高可以改变（如：input）

CSS可以改变标签属性：

  • display : none 隐藏标签
  • display : block 让标签变成块级标签
  • display : inline 让标签变成行内标签
  • display : inline-block 让标签变成行内-块级标签

Html 常见区别

单标签与双标签（容器标签）

  • 只要是首尾呼应的标签，需要包含内容的 都是双标签（容器标签，如： h div span body a p标签），其它的都是单标签，单标签（如：hr br input）无需在里面再放内容

href和src区别

  • href 引用，相当于import，资源对于标签来说不是必须的
  • src 引入，就是这个资源对于标签来说是必须的，如果这个标签缺少这个资源就转不起来

列表标签可继承：

选择器优先级

  1. 相同类型选择器遵循就近原则，叠加原则
  2. 一般不同类型选择器遵循优先级别：id选择器>类选择器>标签选择器

权值：

  • 通配选择符（*）：0
  • 标签：1
  • 类：10
  • 属性：10
  • 伪类：10
  • 伪元素：10
  • id：100
  • important：1000

选择器权值加在一起，大的优先；如果权值相同，后定义的优先。行内元素优先级别永远最高
，除了important 的属性。

CSS布局常用属性

默认情况下，所有的网页标签都在标准流布局中（ 从上到下，从左到右）。想要改变标准流，就需要用到CSS的布局属性，float 和 position。

float

使用float后，块级标签会被迫变成 行内块级标签，注意Float和display的区别

  • float: left 脱离标准流，浮动到父标签最左边
  • float: right 脱离标准流，浮动到父标签最右边

postion

当让某个子标签浮动于父标签任何一个位置，而不会影响到其他子标签的布局的时候，就可以使用postion这样一个属性，而使用了margin这个布局，需要考虑到其他子标签的情况

注意：使用了position后，符合使用float后，脱离标准流的特征，就是块级标签变成行内块级标签。

margin与position的absolute、relative这三个好像都可以定位，什么时候该用margin来定位，又是什么时候用position来定位比较好呢，这三者定位元素的区别在哪？

margin会影响其它元素的布局，一般不用它作定位，它是以相对另外一个元素的距离为单位。position 则不会，相反它才是做定位的，它是以文档左上角为原点的绝对距离为单位。

注意和margin的区别：margin也提供了布局的功能。

position属性常有的值：

值 | 描述
|—-|—-|
absolute | 生成绝对定位元素，相对于static定位以外的第一个父元素进行定位 ,元素通过left，top，right，bottom属性进行规定
fixed | 生成绝对定位的元素，相对于浏览器窗口定位，元素通过left，top，right，bottom属性进行规定
relative | 生成相对的元素，相对于其正常的位置进行定位
static | 默认值，没有定位，元素出现在正常流中（忽略：top，left，bottom，right）
inherit | 规定从父元素继承postion的值

box-sizing

不想额外的扩展盒子的大小，就想它往里走

应用(垂直居中，水平居中)

text-align:center; //能让 行内标签和行内块级标签 以及所有标签内容水平居中
但是不能够让块级标签居中，如div标签，如果想让块级标签div居中,可以用margin属性

所有标签水平居中：
1.行内标签 和 行内块级标签 ： 在父标签中设置 text-align:center；
2.块级标签：在自身设置 margin : 0 auto;

所有标签垂直居中：
1.行内标签 和 行内块级标签 ： 在父标签中设置 line-height；
2.块级标签：在自身设置 postion: absolute; left : 50%; right : 50%; transform:translate(-50%,-50%);

JS的语言特点

html是弱类型语言，是解释执行，非编译运行，所以代码会从上往下走，所以要注意 Script标签放在头部和尾部，大部分人把它放在尾部。

window

  1. 所有全局变量都是window属性
  2. 所有全局函数都是window方法
  3. 动态跳转

document

  1. 可以动态的获取到网页内的所有标签(节点)
  2. 可以对获取到的标签进行 增删改查 (CRUD)

开发思路

首先，分析页面，搭建对应的div骨架，给每个div骨架设置一个相应的Class类，用于CSS渲染，
