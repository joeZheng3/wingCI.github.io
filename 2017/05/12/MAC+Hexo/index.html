HEXO简介

  • Hexo是一款基于Node.js的静态博客生成程序，作者是台湾的tommy35.

环境准备

Homebrew安装

  • ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Node/Npm安装

  • Homebrew安装：brew install node
  • node -v:查看node版本
    npm -v：查看npm版本

Git安装

  • 图形化安装：http://sourceforge.net/projects/git-osx-installer/
  • Homebrew安装：brew install git

Hexo安装

  • sudo npm install -g hexo（-g:全局安装）

提示：官网上的安装 npm install -g hexo-cli，实际安装需要权限sudo

Hexo使用

经过上面操作后，使用浏览器，打开网址http://localhost:4000 ，有Hexo页面显示说明安装启动Hexo服务成功。

Hexo常见命令

  • hexo g （generate 生成静态文件）
  • hexo s （server 启动服务器）
  • hexo d （deploy 部署网站，部署网站前，需要预先生成静态文件）
  • hexo clean （clean 清除缓存文件 (db.json) 和已生成的静态文件 (public)）
